### 2. 加密包格式
```typescript
interface EncryptedPackage {
  version: string;           // BMail 协议版本 (2.0)
  algorithm: string;         // 加密算法 (AES-256-GCM)
  encryptedContent: string;  // 加密后的内容(Base64)
  metadata: {
    timestamp: number;       // 加密时间戳
    contentType: string;     // 内容类型(html)
  };
}
```
