## 实现内容

### 1. 创建 sender.ts 模块
- ✅ 创建 `src/compose/sender.ts` 文件
- ✅ 实现 `Sender` 类,封装完整发送流程
- ✅ 实现所有必需接口:
  - `send()` - 执行完整发送流程
  - `validateInput()` - 验证输入数据
  - `buildSendMessage()` - 构建发送消息
  - `sendToHost()` - 发送到宿主页面并等待响应

### 2. 发送流程
完整的发送流程包括:
1. 验证输入(收件人、主题、内容)
2. 调用 Encryptor 加密内容
3. 构建 SEND_EMAIL 消息
4. 通过 postMessage 发送到宿主
5. 等待 SEND_EMAIL_RESPONSE 响应(30秒超时)
6. 返回发送结果

### 3. UI 实现
- ✅ 添加主题输入框
- ✅ 添加发送状态覆盖层
- ✅ 三种状态显示:
  - loading: 发送中(⏳)
  - success: 发送成功(✓)
  - error: 发送失败(✗)
- ✅ 状态自动隐藏(3秒后)

### 4. 集成到 compose.ts
- ✅ 导入 `Sender` 类
- ✅ 实现 `showSendStatus()` 函数
- ✅ 实现 `clearForm()` 函数
- ✅ 重写发送按钮逻辑:
  - 获取主题、收件人、内容
  - 禁用发送按钮
  - 显示发送状态
  - 调用 Sender.send()
  - 处理发送结果
  - 成功后清空表单
  - 恢复发送按钮

### 5. 消息协议
```typescript
// Iframe → 宿主
{
  type: 'SEND_EMAIL',
  payload: {
    recipients: string[],
    subject: string,
    encryptedContent: string,  // 序列化的加密包
    metadata: {
      timestamp: number,
      version: string
    }
  }
}

// 宿主 → Iframe
{
  type: 'SEND_EMAIL_RESPONSE',
  payload: {
    success: boolean,
    error?: string,
    messageId?: string
  }
}
```

## 完整发送流程

用户操作:
1. 添加收件人
2. 输入邮件主题
3. 撰写邮件内容
4. 点击发送按钮

系统处理:
1. 验证收件人、主题、内容
2. 禁用发送按钮
3. 显示"正在发送邮件..."
4. 调用 Sender.send():
   - 验证输入
   - 加密内容
   - 构建消息
   - 发送到宿主
   - 等待响应
5. 处理响应:
   - 成功: 显示成功提示,清空表单
   - 失败: 显示错误提示
6. 恢复发送按钮
